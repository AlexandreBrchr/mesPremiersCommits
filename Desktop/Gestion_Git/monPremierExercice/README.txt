Présentation du projet

1.Objectif
Mise en pratique des notions acquises dans les deux premières parties du cours Git&GitHub.

2. Travail à réaliser 
	- Création d'un fichier "README.txt" présentant le projet,
	- Au moins 2 autres fichiers, 
	- Au moins 4 commits distincts montrant l'ajout puis l'ajout de certains fichiers.

3. Régles à respecter
	- Prêter attention à la qualité des messages de commit,
	- Envoyer le repository Git sur GitHbub lorsque celui-ci est créé,
	- Envoyer votre travail en compressant LE document texte, préalablement renseigné de l'URL du repository GitHub.

4. Astuce
Pour compresser un fichier :
	1. Sélectionner le fichier à compresser 
	2. Cliquer avec le bouton droit de votre souris 
	3. Cliquer sur "Envoyer  vers"
	4. Selectionner "Dossier compressé" 
	5. Le fichier zippé est créé
